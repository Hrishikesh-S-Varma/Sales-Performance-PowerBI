## Project Overview
This project analyzes company sales performance across regions, products, and time periods using an interactive Power BI dashboard.

## Business Problem
Sales teams need clear visibility into revenue trends, top-performing products, and underperforming regions to make data-driven decisions.

## Key KPIs
- Total Revenue
- Sales Growth %
- Average Order Value
- Top Products by Revenue
- Regional Sales Contribution

## Dashboard Features
- Year & Region slicers
- Trend analysis using line charts
- Product-wise and region-wise comparisons
- High-level KPI cards for executive overview

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
- GitHub

## Outcome
The dashboard enables stakeholders to quickly identify sales trends, growth opportunities, and areas requiring corrective action.
